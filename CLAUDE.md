# metacat

## Agent skills

### Backlog

Issues live in GitHub Issues for this repo. Use the `gh` CLI for all backlog operations. See `docs/agents/backlog.md`.

### Triage labels

The five canonical triage roles map 1:1 to label strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` and one `docs/adr/` at the repo root. See `docs/agents/domain.md`.
